# Pharmacy Micro-Service

## Running the Project

### Requirements

1. NodeJS >= 14
2. TypeScript

## Commands

To install the dependencies, run:

```
yarn
```

Once the dependencies are installed, you need to create a `.env` file on the project's root. It's format should be the following:

```
DB_USER='foo'
DB_PASS='bar'
DB_NAME='baz'
DB_HOST=0.0.0.0
DB_PORT=0000
```

In order to

- Running it locally

```
yarn start
```
